# Clone Gotchas

- **Cloning aesthetics without understanding**: Copying visual style without understanding information architecture. Serve YOUR users, not the original's.
- **Responsive breakpoint guessing**: Mobile behavior doesn't map cleanly to standard sm/md/lg breakpoints.
- **Design token compliance shortcuts**: Hardcoded hex colors and pixel values instead of design system tokens.
- **Placeholder realism**: "Lorem ipsum" in a hero that needs real copy to evaluate. Use realistic text.
- **Type safety in generated components**: Generated JSX may not be TypeScript-compatible. Run type checks.
