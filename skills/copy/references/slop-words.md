# Banned Words — The Slop List

Every word on this list is banned from copy output. Each entry explains WHY it's banned so you know the principle, not just the rule.

## Hype words (claim without evidence)

| Word | Why it's banned |
|------|----------------|
| revolutionary | Nothing you're building is revolutionary. It's useful or it's not. |
| cutting-edge | Meaningless. What edge? Compared to what? |
| game-changing | If you have to say it, it isn't. |
| next-generation | Generation of what? Vague futurism. |
| disruptive | Only applies retrospectively. You can't call yourself disruptive. |
| innovative | The word "innovative" is never innovative. Show the innovation. |
| best-in-class | Class of what? Compared to whom? Self-awarded superlatives. |
| world-class | See best-in-class. Same problem. |
| state-of-the-art | Claims expertise without demonstrating it. |
| bleeding-edge | Synonym for "unstable" dressed up as a feature. |
| groundbreaking | You're not breaking ground. You're writing software. |

## Filler words (add nothing)

| Word | Why it's banned |
|------|----------------|
| seamlessly | Nothing is seamless. Name what friction was removed instead. |
| robust | Robust how? Against what? This word replaces specificity with vibes. |
| holistic | Means "we do multiple things" without naming them. |
| scalable | Unless you have benchmarks, this is a promise, not a feature. |
| comprehensive | "We have a lot of stuff" is not a value prop. |
| streamline | Vague verb. What specifically gets faster or simpler? |
| optimize | Same problem as streamline. Optimize what, by how much? |

## Power verbs that mean nothing

| Word | Why it's banned |
|------|----------------|
| leverage | Corporate jargon for "use." Just say "use." |
| unlock | Implies hidden value that was always there. Usually a stretch. |
| empower | Patronizing. People don't need empowering. They need tools that work. |
| transform | Transformation requires before/after evidence. Without it, it's hype. |
| supercharge | Nobody's product is a battery. |
| elevate | Vague upward motion. Elevate what? From where? To where? |
| synergy | The most mocked word in business for a reason. |
| accelerate | Unless you have speed benchmarks, this is a guess. |

## Lazy phrases

| Phrase | Why it's banned |
|--------|----------------|
| actionable insights | Every insight should be actionable. The phrase adds nothing. |
| deep dive | You're reading text, not scuba diving. Say "detailed analysis." |
| move the needle | What needle? By how much? Metaphor replacing measurement. |
| low-hanging fruit | Dismisses work as easy. Usually the "easy" thing isn't. |
| at the end of the day | Filler. Delete and the sentence is better. |
| take it to the next level | What level? This is a video game metaphor, not a value prop. |
| save time and money | Could describe any tool ever made. Be specific about how much and how. |
| improve your workflow | What workflow? What improves? Generic to the point of meaningless. |
| boost efficiency | By how much? Compared to what? Vague performance claim. |

## The principle

Every banned word shares the same flaw: it replaces a specific claim with a vague one. The fix is always the same: state the specific thing. What exactly changes? By how much? Compared to what?

If you can't answer those questions, the copy isn't ready. The problem isn't the word — it's that you don't know the specific yet.
