# Inversion Thinking

Flip every assumption. The opposite of the obvious answer is often the real answer.

## Method
1. Write the current assumption (e.g., "users need onboarding")
2. Invert it ("users need NO onboarding")
3. Ask: what product would make the inversion true?
4. Repeat for each clause in the value hypothesis

## Applied to value hypothesis
Take each piece and reverse it:
- "solo founder" → what if teams? what if non-technical?
- "measurably better" → what if immeasurable but obviously better? what if worse but learning?
- "every session" → what if once a month? what if continuous?
- "less effort" → what if MORE effort but 10x the result?

## Output
5-10 inversions. Each is a potential pivot direction. Most are bad. One might be the real product.
