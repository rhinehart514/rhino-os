Eval System

Three-Tier Eval Architecture — Ungameable by Design

1
Computed — Fully Mechanical (0 LLM)
Color contrast (WCAG formula) · Font hierarchy (CSS computed) · Click target sizes (min 44×44px) · Element density (per viewport) · Whitespace ratio · CLS score (Playwright native) · Route graph depth
UNGAMEABLE — pure math, no model in the loop
2
Behavioral — Blind Agent Task Completion (val_bpb equivalent)
Playwright launches app as fresh user · Blind LLM (zero app context) attempts primary task · Metric: pass/fail + time + click count · The app must actually work for a naive user
HARD TO GAME — external agent, real task, binary outcome
3
Corpus Comparison — Anchored LLM Gap Analysis
Show Claude: current output + 3-5 corpus examples · Ask for specific gap analysis, not rating · Multi-model consensus required (Claude + GPT-4V + Gemini) · ELO ranking not absolute score · Gaming 3 models simultaneously is effectively impossible
RESISTANT — consensus + real examples as anchor
