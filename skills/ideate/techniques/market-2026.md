# 2026 Market Context

What's true RIGHT NOW that changes what we should build. Update this as the market shifts.

## The 2026 landscape (verified signals)

### AI agents are the platform, not the feature
- Salesforce Agentforce: $540M ARR, 18,500 customers (fastest-growing product)
- Every SaaS company shipping "AI agents" — most are wrappers, few are real
- MCP (Model Context Protocol) winning as the integration standard
- Claude Code marketplace: 340+ plugins, 89K installs on top skill
- The gap: everyone builds agents that WRITE code. Nobody builds agents that THINK about products.

### Verification > Generation
- Anthropic's own guide: verification skills are "#1 worth spending a week on"
- Agentic coding trends: "verification lattice" (gate-based state machines) is top pattern
- Amplitude shipped 5 monitoring agents (Feb 2026) — continuous, not point-in-time
- The shift: 2024 was "AI generates code." 2026 is "AI proves the code works."

### Solo founders are the new normal
- AI tools let one person do what required 5-10 in 2024
- YC funding solo founders more often (signal from W26 batch)
- "Cursor for PMs" demand is real — no product fills it
- Solo founder needs: measurement, strategy, customer dev, pricing — not just code

### What's being disrupted RIGHT NOW
- Product analytics (Amplitude going agent-native)
- CI/CD monitoring (Faros, LinearB adding intelligence layer)
- Customer research tools (Looppanel, BuildBetter — but none code-integrated)
- Engineering metrics (DORA metrics being automated)
- Product management tools (Linear, Notion — but none measure value)

### Pricing in 2026
- SaaS fatigue is real — founders resist new subscriptions
- Usage-based pricing gaining (pay for what you use)
- Free tier expected — the question is what's behind the paywall
- Claude Code plugins are currently free — marketplace monetization coming
- $19-49/mo is the solo founder sweet spot (Linear $8, Notion $10, Vercel $20)

## How to use this in ideation

### "Why now?" answers that work in 2026
- "MCP just became the standard, so we can connect to any tool" → integration ideas
- "Claude Code marketplace just launched, 89K users on top plugin" → distribution ideas
- "Verification lattice is the emerging pattern" → quality/testing ideas
- "Solo founders can now build what teams built" → single-player tool ideas
- "Every SaaS is shipping AI agents but none measure value" → measurement ideas

### "Why now?" answers that DON'T work
- "AI is getting better" → always true, not specific
- "Developers need help" → always true since forever
- "The market is big" → says nothing about timing

### Market gaps to ideate against
1. **No product-thinking tool for solo founders** — Linear manages tasks, Notion manages docs, nothing manages "is this the right thing to build?"
2. **No verification-native development** — CI runs tests AFTER merge. Nothing runs assertions DURING development.
3. **No learning loop in any dev tool** — every tool is stateless session-to-session. Nobody tracks predictions, grades them, and updates a model.
4. **No "product score"** — code quality tools exist (SonarQube, CodeClimate). Product quality tools don't.
5. **No agent coordination layer** — MCP connects tools. Nothing coordinates the agents that USE those tools.

### Competitors and positioning
- **Cursor/Windsurf**: Code generation. We don't compete — we're the layer ABOVE code.
- **Linear/Notion**: Project management. We don't manage tasks — we measure value.
- **PostHog/Amplitude**: Analytics. They need users + data. We work from day 0.
- **SonarQube/CodeClimate**: Code quality. They measure code. We measure product.
- **Nobody**: Product measurement + learning loop + agent coordination for solo founders.
