# 2026 Market Context

What's true RIGHT NOW that changes what to build. Strategic lens, not just data.

## The landscape (verified signals)

### AI agents are the platform
- Salesforce Agentforce: $540M ARR, 18,500 customers (fastest-growing product)
- Every SaaS company shipping "AI agents" — most are wrappers, few deliver real value
- MCP (Model Context Protocol) winning as the integration standard
- Claude Code marketplace: 340+ plugins, 89K installs on top skill
- The gap: everyone builds agents that WRITE code. Nobody builds agents that THINK about products.

### Verification > Generation
- Anthropic's own guide: verification skills are "#1 worth spending a week on"
- Agentic coding trends: "verification lattice" (gate-based state machines) is top pattern
- Amplitude shipped 5 monitoring agents (Feb 2026) — continuous, not point-in-time
- The shift: 2024 was "AI generates code." 2026 is "AI proves the code works."

### Solo founders are the new normal
- AI tools let one person do what required 5-10 in 2024
- YC funding solo founders more often (signal from W26 batch)
- "Cursor for PMs" demand is real — no product fills it
- Solo founder needs: measurement, strategy, customer dev, pricing — not just code

### Saturated categories (avoid)
- Coding assistants / copilots (ChatGPT 62-81% share, Copilot dominant)
- Consumer chatbots (diminishing returns from scaling)
- Content generation (commodity, race to zero)
- Horizontal AI agents (half the entire 2000+ company market)
- Marketing automation (table stakes, no differentiation left)

### Emerging categories (opportunities)
- Vertical AI agents (industry-specific: legal, healthcare, finance)
- AI evaluation/measurement infrastructure (enterprises can't measure ROI)
- Founder decision tools (YC 2026: "what to build, not how")
- Boring industry AI (YC: metal mills, government, field service)
- AI agencies with software margins (deliver finished work, not tools)
- Physical AI / vocational coaching (multimodal + wearables)
- Financial agent swarms (new quant strategies)

### What's being disrupted NOW
- Product analytics (Amplitude going agent-native)
- CI/CD monitoring (Faros, LinearB adding intelligence layer)
- Customer research tools (Looppanel, BuildBetter — but none code-integrated)
- Engineering metrics (DORA metrics being automated)
- Product management tools (Linear, Notion — but none measure value)

## Pricing in 2026
- SaaS fatigue is real — founders resist new subscriptions
- Usage-based pricing gaining (pay for what you use)
- Free tier expected — the question is what's behind the paywall
- Claude Code plugins are currently free — marketplace monetization coming
- $19-49/mo is the solo founder sweet spot (Linear $8, Notion $10, Vercel $20)

## Strategic implications

### For positioning
- Don't compete with code generation tools — compete with product thinking gaps
- "Measurement" is the wedge, but "product intelligence" is the category
- Distribution through Claude Code marketplace is the cheapest channel available right now
- Solo founders are the beachhead user — they have all the problems and no team to delegate to

### For timing
- MCP adoption window: 6-12 months before consolidation
- Claude Code plugin marketplace: early enough to establish, growing fast enough to matter
- Verification-native tools: no established player yet
- Risk: Anthropic builds this themselves (high risk, 12-18 month window)

### For competitive response
- Default: "keep building." Override only with strong evidence.
- Someone launches similar: check if they have distribution. If no, irrelevant. If yes, differentiate or accelerate.
- Big tech enters space: focus on wedge use case they'll never prioritize. Solo founders are too small for Salesforce.
