# Divergent Thinking

Generate 20+ ideas with NO filter. Quantity over quality. The weird ones are the point.

## Method
1. Start with the prompt (feature, problem, direction)
2. Set a target: 20 ideas minimum
3. Use these triggers if stuck:
   - **SCAMPER**: Substitute, Combine, Adapt, Modify, Put to other use, Eliminate, Reverse
   - **Random word**: Pick a random object. Force a connection to the problem.
   - **Worst idea**: What's the absolute worst solution? Now invert it.
   - **10x constraint**: What if it had to serve 10x the users? 1/10th the users?
   - **Different medium**: What if this was a physical product? A game? A service?
4. NO evaluation during generation. Evaluation comes after.

## Output
Raw numbered list. No filtering. Mark any that spark something with *.
