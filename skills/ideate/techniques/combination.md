# Combination Thinking

Smash two things together. Innovation is mostly recombination.

## Combination types
- **Feature + Feature**: /eval + /taste = visual evaluation with scores
- **Product + Channel**: rhino-os + Slack = product health in your team chat
- **Tool + Market**: scoring + agencies = client project quality reports
- **Skill + External**: /ship + Vercel = one-command scored deployment
- **Data + Action**: predictions.tsv + GitHub Issues = auto-created issues from failed predictions

## Method
1. List 10 things (features, tools, channels, markets)
2. Randomly pair them (use a grid or just pick)
3. For each pair: what new thing would this create?
4. Most combinations are garbage. 1-2 will be interesting.

## Output
5-8 combinations. Rate each: garbage / interesting / build this.
