# Taste Dimensions

## Gate Dimensions (cap overall at 30 if <30)
- **layout_coherence**: Does the layout have consistent alignment, grid structure, and spacing logic? Score 0 if elements feel randomly placed.
- **information_architecture**: Is content organized in a way that matches the user's mental model? Routes, navigation, hierarchy.

## Core Dimensions
- **hierarchy**: Visual weight distribution. Is there a clear primary element on each screen? One primary action per view.
- **breathing_room**: Whitespace usage. Cramped = low score. Generous spacing that aids comprehension = high.
- **contrast**: Color and weight contrast between elements. Can you distinguish sections? Can you find the CTA?
- **polish**: Pixel-level refinement. Consistent border radius, aligned icons, proper line heights.
- **emotional_tone**: Does the visual design match the product's personality? Serious/playful/professional/warm.
- **distinctiveness**: Would you recognize this product in a lineup? Or does it look like every other app?
- **scroll_experience**: Below-the-fold content rewards scrolling. Not just more of the same.
- **wayfinding**: Can you get back? Can you find any page from main nav? Breadcrumbs, active states.
- **information_density**: Right amount. Too much = overwhelming. Too little = empty. Match density to task.

## Scoring
- 0-20: Broken or actively harmful
- 20-40: Clearly below average
- 40-60: Average, nothing notable
- 60-80: Above average, intentional design
- 80-100: Exceptional, would show as example
