# Configuration Gotchas

- **Config to fix scores**: Scores are low because the product needs work, not because configuration is wrong.
- **Economy to save money**: If eval scores are low, economy mode makes builder output worse. Economy is for exploration.
- **Full-auto to skip friction**: The hard gate exists because "obvious" moves have highest skip-regret. Fix the plan, not the gate.
- **Preferences override escalation**: If you're overriding 5+ settings, the defaults are wrong — fix defaults instead.
